Skills & Coursework

AUCSC 111- Intro to Computational Thinking and Problem Solving (Python)

An introduction to computational thinking, problem solving, and the fundamental ideas of computing. Topics include algorithms, abstraction, and modelling; the syntax and semantics of a high-level language (e.g. Python); fundamental programming concepts and data structures, including simple containers (variables, arrays, lists, strings, dictionaries); sequencing, conditionals and repetition; documentation and style; object-oriented programming; exceptions and error handling; recursion; simple algorithm analysis and run- time efficiency.

AUCSC 112 (Java+Algorithm)

An introduction to object-oriented design and programming; algorithm analysis; data structures and container classes including arrays, lists, arraylists, stacks, queues, user-defined structures and an introduction to trees, along with associated algorithms such as insertion, removal, sorting and retrieval; introduction to recursion

CMPUT 272 - Formal Systems and Logic in Computing Science(Algorithm+Psudocode)

An introduction to the tools of set theory, logic, and induction, and their use in the practice of reasoning about algorithms and programs. Basic set theory; the notion of a function; counting; propositional and predicate logic and their proof systems; inductive definitions and proofs by induction; program specification and correctness

CMPUT 201 - Practical Programming Methodology (C)

Fundamental principles of software engineering based on abstract data types and their implementations. Students use C and software development tools of the Unix environment.

CMPUT 204 - Algorithms

Algorithm design and analysis, with emphasis on fundamentals of searching, sorting, and graph algorithms. Examples include divide and conquer, dynamic programming, greedy methods, backtracking, and local search methods, together with analysis techniques to estimate program efficiency.

CMPUT 229 - Computer Organization and Architecture (MIPS)

Number representation, computer architecture and organization, instruction-set architecture, assembly-level programming, procedures, stack frames, memory access through pointers, exception handling, computer arithmetic, floating-point representation, datapath, control logic, pipelining, memory hierarchy, virtual memory.

CMPUT 291 - File and Database Management (SQL & MongoDB)

Concepts in computer data organization and information processing; entity-relationship model; relational model; SQL and other relational query languages; storage architecture; physical organization of data; access methods for relational data.

CMPUT 340 - Introduction to Numerical Methods(MATLAB)

Computer arithmetic and errors. The study of computational methods for solving problems in linear algebra, non-linear equations, optimization, interpolation and approximation, and integration. This course will provide a basic foundation in numerical methods that supports further study in machine learning; computer graphics, vision and multimedia; robotics; and other topics in Science and Engineering.

CMPUT 301 - Software Engineering (Java - Android Studio)

Software process; revision control; Unified Modeling Language (UML); requirements; software architecture, design patterns, frameworks, design guidelines; unit testing; refactoring; software tools.

CMPUT 366 - Search and Planning in Artificial Intelligence (Python)

Algorithms for solving deterministic shortest path problems with factored and non-factored states, combinatorial optimization problems, constraint satisfaction problems, and multi- agent problems.

Courses On Process

CMPUT 355 - Games, Puzzles, Algorithms (Python, C, C++, Java)

An introduction to algorithms and theory behind computer programs that solve puzzles (mazes, peg solitaire, etc.) or play games (chess, Go, Hex, etc.). This course is intended for a general audience.

CMPUT 455 - Search, Knowledge and Simulation (Python+Algorithms)

When making decisions in games, computers rely on three main ideas: search, knowledge and simulations. Knowledge can be created by machine learning techniques and encoded in deep neural networks. Search and simulations help to understand the short and long-term consequences of possible actions. This course leads from basic concepts to state-of-the-art decision-making algorithms

CMPUT 404 - Web Applications and Architecture (HTML,CSS,HTTP,TCP,CGI,JAVASCRIPT,XML,WEB MINING, REST, AJAX, WEBSOCKETS)

Introduction to modern web architecture, from user-facing applications to machine-facing web-services. Topics include: the evolution of the Internet, relevant technologies and protocols, the architecture of modern web-based information systems, web data exchange and serialization, and service-oriented middleware.
