Functionality

Social Distribution App is a dynamic platform where users can create diverse types of posts that can be made public or exclusive to friends. Users can easily connect with their friends, view and interact with their posts, and manage their friend requests. This app provides an intuitive interface for users to like, comment, like comment, and share on posts, promoting social engagement. All the applications within the app are seamlessly interconnected through API calls, making it easy for users to navigate and stay updated. Additionally, the inbox feature is a handy tool that provides notifications for post inbox, comment post inbox, like comment, and like comment inbox, ensuring that users never miss an important update.
